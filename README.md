## Editing the site

Everything is in `index.html` — one file, no build tools needed.

**To change text content**, search for the section you want (e.g. `id="about"`) and edit the HTML directly.

**To change colours**, find the `:root` block near the top of the `<style>` tag:

```css
:root {
  --void:      #0A0806;   /* page background   */
  --bone:      #E8DFD0;   /* main text colour  */
  --parchment: #C9BDA8;   /* accent colour     */
  --ash:       #6B6560;   /* muted text        */
  --ghost:     #1C1814;   /* card backgrounds  */
}
```

**To update your email**, find `mailto:your@email.com` and replace with your real address.

**To update social links**, find the `.socials` section and replace the `href="#"` values.
